# Project Memory Index

Per-project working knowledge and the cross-project master plan.

| Path | Purpose |
|---|---|
| `project-master-plan/` | Cross-project roadmap & optimization (e.g. `optimization-roadmap.md`) |
| `projects/<ProjectName>/` | One folder per project (see structure below) |

## Per-Project Structure (`projects/<ProjectName>/`)

- `goals.md` — objectives & success criteria
- `stack.md` — technologies, versions, rationale
- `conventions.md` — coding conventions, patterns
- `todos.md` — active task list
- `adrs/` — Architecture Decision Records
- `<name>-xs.md … -xl.md` — tiered project context
